# Changelog

## 1.0.1

- Removed dynamic Python execution from custom function processing for QGIS plugin security validation.
- Routed bundled standard custom functions through fixed internal handlers.

- Initial koji DataQuery plugin structure.
- Added CSV and GeoPackage preprocessing tools.
- Added CSV aggregation query tool.
- Added CSV and GeoPackage LEFT JOIN/UNION tools.
- Added project-based preset/config storage.
- Added QGIS plugin ZIP build, validation, and upload documentation.
